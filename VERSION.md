# Czas Client - Wersja i Informacje

## Wersja Moda
- **Wersja:** 1.0.0
- **Autor:** 348 s
- **Data wydania:** 2026-03-12

## Silnik i Technologia
- **Silnik:** Minecraft Forge
- **Wersja Forge:** 47.3.0 (dla Minecraft 1.20.1)
- **Wersja Java:** 17+
- **Język programowania:** Java
- **System budowania:** Gradle 8.1.1

## Kompatybilność
- **Minecraft:** 1.18.2 - 1.21.4
- **Forge:** 40.0.0+
- **Java:** 17 lub nowsza

## Funkcje
- **Movement:** While Speed, Fake Creative, Spider, Long Jump, No Fall, Scaffold
- **Combat:** Aim Assist, Auto Clicker, No Slow, Nuker
- **Utilities:** Auto Regen, Inventory Walk, Auto Armor
- **World:** Fast Break, Timer
- **Render:** Full Bright

## GUI
- **Otwieranie:** Prawy Shift
- **Logo:** Czerwona zegar
- **Motywy:** Red/Black, Blue, Green
- **Języki:** Polski, Angielski

## Kompilacja
```bash
# Lokalna kompilacja
./gradlew build
./gradlew shadowJar

# GitHub Actions
```

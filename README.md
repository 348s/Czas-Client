# Czas Client

A comprehensive Minecraft client by 348 s with configurable mods and enhanced gameplay features.

## Features

### Movement Hacks
- **While Speed** - Increases movement speed
- **Fake Creative** - Grants creative-like abilities
- **Spider** - Allows climbing walls
- **Long Jump** - Enhanced jump height/distance
- **No Fall** - Prevents fall damage
- **Scaffold** - Auto-places blocks under player

### Combat Hacks
- **Aim Assist** - Helps aim at entities
- **Auto Clicker** - Automatic clicking
- **No Slow** - Prevents movement slowdown

### Utility Hacks
- **Nuker** - Auto-breaks blocks
- **Auto Regen** - Auto health regeneration
- **Inventory Walk** - Walk with inventory open
- **Auto Armor** - Auto-equips best armor

### World Hacks
- **Fast Break** - Increases breaking speed
- **Timer** - Modifies game speed

### Render Hacks
- **Full Bright** - Increases world brightness

## GUI & Configuration

- **Right Shift** - Toggle ClickGUI
- **Red Clock Logo** - Client branding
- **Customizable Keybinds** - Right-click modules to bind
- **Gradient Themes** - Red/Black, Blue, Green
- **Language Support** - English/Polish
- **Configuration System** - Save/load settings

## Compatibility

Compatible with Minecraft versions 1.18.2 to 1.21.4

## Installation

1. Build the project using Gradle
2. Place the generated JAR in your mods folder
3. Launch Minecraft with Forge

## Usage

- Press **Right Shift** to open the ClickGUI
- Left-click modules to toggle
- Right-click modules to change keybinds
- Configure settings through the GUI
